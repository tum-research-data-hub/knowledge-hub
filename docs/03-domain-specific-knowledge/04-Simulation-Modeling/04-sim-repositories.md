# **Simulation & Modeling Repositories**

Sharing simulation data, models, and benchmarks is essential for reproducibility and collaborative development. This page lists repositories and platforms relevant to simulation and modeling research.

---

## **Simulation Data Repositories**

| Repository | Description | Link |
|------------|-------------|------|
| **Zenodo** | General-purpose research data repository (CERN), widely used for simulation datasets and benchmarks | [zenodo.org](https://zenodo.org/) |
| **DaRUS (TU Munich)** | TUM's institutional data repository based on Dataverse | [darus.uni-stuttgart.de](https://darus.uni-stuttgart.de/) |
| **Fordatis (FIZ Karlsruhe)** | Research data repository for engineering and natural sciences | [fordatis.fraunhofer.de](https://fordatis.fraunhofer.de/) |

---

## **Benchmark and Validation Databases**

| Repository | Description | Link |
|------------|-------------|------|
| **NASA Turbulence Modeling Resource** | Validation cases for CFD turbulence models | [turbmodels.larc.nasa.gov](https://turbmodels.larc.nasa.gov/) |
| **ERCOFTAC** | European Research Community on Flow, Turbulence, and Combustion — quality and trust in CFD | [ercoftac.org](https://www.ercoftac.org/) |
| **NIST Benchmark Problems** | Reference problems for FEA and multiphysics verification | [nist.gov](https://www.nist.gov/) |
| **OpenFOAM Tutorials** | Built-in tutorial cases that serve as community benchmarks | [openfoam.com/documentation/tutorial-guide](https://www.openfoam.com/documentation/tutorial-guide) |

| Repository | Description | Link |
